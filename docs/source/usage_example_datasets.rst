Example datasets
================

The repository already ships with a set of small example files used by the
interactive scripts in ``user_scripts/``.

In addition, a dedicated, packaged example-dataset release is planned for the
future so that users can download a curated set of public demonstration stacks
independently of the source repository.

Planned contents for the example-data page
------------------------------------------

This page will later document:

- which example datasets are bundled directly with the repository,
- which larger public datasets can be downloaded separately,
- how each dataset maps to the tutorial scripts,
- and which datasets are synthetic versus derived from public benchmark data.


For now
-------

For now, the practical starting points are:

- ``example_data/PICASSO_examples/`` for public small example stacks,
- the synthetic example used in
  :doc:`usage_unmix_full_tzcyx_synthetic_example`,
- and the interactive tutorial scripts in ``user_scripts/``.
